## ✨ Project Vision
**Rent a Ride** is a high-performance, full-stack car rental platform designed for a "Midnight Dark" premium experience. Built with a focus on cinematic aesthetics and industrial-grade stability, it bridges the gap between luxury fleet providers and elite travelers.

## 🧠 Core Architecture
The system is divided into three distinct orchestration layers, each tailored for specific operational workflows:

### 👤 User Experience (Client)
- **Cinematic Discovery**: Ken Burns effect parallax hero section with integrated floating glassmorphism search.
- **Precision Booking**: Multi-step checkout wizard with secure billing and real-time availability validation.
- **Smart Logistics**: Dynamic location selector with district-to-city mapping.
- **Financial Security**: 256-bit encrypted transactions via Razorpay 3.0.

### 🛡️ Administrative Command (Admin)
- **Fleet Control**: Full CRUD operations with instant deployment to the marketplace.
- **Lifecycle Management**: Real-time status tracking (On-Trip, Overdue, Cancelled).
- **Vendor Governance**: KYC-style approval/rejection workflows for third-party partners.

## 🔐 Authentication & Security
- **JWT Protection**: Implementation of Access and Refresh token logic for persistent, secure sessions.
- **RBAC**: Strict route protection ensures sensitive administrative data remains isolated from standard users.
- **OAuth 2.0**: Integrated Google Sign-In for friction-less onboarding.
